# ChocoPy Compiler

How to run:

Install node modules and make package (run only once)

```bash
npm install
make
```

build webpage and run compiler

```bash
npm run build-web
cd build
python -m SimpleHTTPServer
```

Run your ChocoPy codes at ```localhost:8000```
